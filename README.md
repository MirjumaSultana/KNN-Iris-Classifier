# KNN Iris Classifier 🌸

## Overview
This project uses the K-Nearest Neighbors (KNN) algorithm to classify Iris flowers into:
- Setosa
- Versicolor
- Virginica

## Dataset
Built-in Scikit-learn Iris Dataset

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Steps
1. Load Iris dataset
2. Split into training/testing sets
3. Train KNN model
4. Evaluate using accuracy
5. Compare K values

## Best Accuracy
Achieved up to 100% accuracy
